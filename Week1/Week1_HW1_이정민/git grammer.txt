 git init : 저장소를 초기화하여 새로운 저장소를 만든다.

 git config : 설정 내용을 확인하고 변경할 수 있다.

 git help : 명령어에 대한 도움이 필요할 때 사용할 수 있다.

 git status : 저장소 상태를 체크할 수 있다. 

 git add <file name> / git add* : 변경된 파일을 인덱스에 추가할 수 있다.   

 git commit -m : 변경 내용을 확정할 때 사용한다.

 git branch : 새로운 브랜치를 만들어 개발을 진행하고 싶을 때 사용한다.

 git checkout : 체크하고 싶은 저장소로 옮겨갈 때 사용한다.

 git merge : 브랜치에서 작업을 끝내고 병합할 때 사용한다.
 
 git push : 로컬 컴퓨터에서 작업하 내용을 온라인으로도 보고 싶을 때 사용한다.
 
 git pull : 로컬 컴퓨터에서 작업할 때, 작업하고 있는 저장소의 최신 버전을 원하면 이 명령어로 변경사항을 다운로드 할 수 있다.

 git diff <원래 가지> <비교 대상 가지>: 내용을 병합하기 전 브랜치를 비교해 볼 수 있다. 

 git clone : 저장소를 복제할 때 사용한다.
